# 基于K325T的高速任意波形信号发生器 - 设计文档

## 1. 系统概述

### 1.1 项目背景

本作品为第二十一届研电赛优利德赛题二"任意波形信号发生器"的参赛作品。基于正点原子Kintex-7 325T开发平台，利用AD9144高速DAC芯片，实现采样率达2.8GSPS、16bit分辨率的任意波形信号发生器。

#### 2.1.3 DDS波形生成原理

```
┌─────────────────────────────────────────────────────────────┐
│                      DDS架构                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   频率控制字(48bit) ──► ┌──────────┐                        │
│                         │ 相位累加器 │──► 相位值(48bit)       │
│   相位偏移(48bit) ────► │  + 偏移  │                        │
│                         └──────────┘                        │
│                               │                             │
│                               ▼                             │
│                         ┌──────────┐                        │
│                         │ 相位截断 │──► 地址(12bit)          │
│                         │ (高12位) │                        │
│                         └──────────┘                        │
│                               │                             │
│                               ▼                             │
│                         ┌──────────┐                        │
│                         │ 波形LUT  │──► 幅度值(16bit)        │
│                         │ 4096点   │                        │
│                         └──────────┘                        │
│                               │                             │
│                               ▼                             │
│                         ┌──────────┐                        │
│                         │ 幅度缩放 │──► 最终输出             │
│                         │ 偏置调整 │                        │
│                         └──────────┘                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**频率分辨率计算：**
- 相位宽度：48 bit
- 系统时钟：1 GSPS (插值前)
- 频率分辨率 = f_clk / 2^48 = 1e9 / 2.81e14 ≈ 3.55 μHz

**频率控制字公式：**
```
phase_inc = f_out × 2^48 / f_clk
```

**常用频率控制字：**

| 目标频率 | 控制字(十六进制) | 控制字(十进制) |
|----------|------------------|----------------|
| 10 MHz | 0x028F5C28F5C28 | 1,125,899,906,842,624 |
| 50 MHz | 0x0CCCCCCCCCCD | 5,629,499,534,213,120 |
| 100 MHz | 0x19999999999A | 11,258,999,068,426,240 |
| 400 MHz | 0x666666666666 | 45,035,996,273,704,960 |

#### 2.2.3 SPI寄存器配置流程

```
上电初始化顺序：
1. 复位DAC (硬件复位或软复位)
2. 配置LMK04828时钟管理器
3. 配置AD9144基本参数
4. 使能JESD204B链路
5. 检查链路同步状态
6. 配置DAC输出模式
7. 使能DAC输出
```

> **注**：本项目采用单AD9144方案，DAC输出直接通过FMC子卡对外输出，未设计额外模拟前端电路。频率响应和输出幅度通过数字域调节实现。

#### 3.2.2 核心功能模块

**1. 波形控制模块 (`awg_uart_control.py`)**

```python
class AWGController:
    """AWG设备控制器"""

    def __init__(self, port: str, baudrate: int = 115200):
        self.serial = serial.Serial(port, baudrate, timeout=1)

    def set_frequency(self, freq_hz: float):
        """设置输出频率"""
        phase_inc = int(freq_hz * (2**48) / 1e9)
        self.write_reg(0x10, phase_inc >> 16)
        self.write_reg(0x14, phase_inc & 0xFFFF)
        self.write_reg(0x2C, 1)  # APPLY

    def set_amplitude(self, amplitude: float):
        """设置输出幅度 (0.0 ~ 1.0)"""
        amp_q15 = int(amplitude * 32767)
        self.write_reg(0x20, amp_q15)
        self.write_reg(0x2C, 1)  # APPLY

    def set_waveform(self, mode: int):
        """设置波形模式"""
        self.write_reg(0x28, mode)
        self.write_reg(0x2C, 1)  # APPLY

    def read_status(self) -> dict:
        """读取设备状态"""
        return {
            'id': self.read_reg(0x00),
            'version': self.read_reg(0x04),
            'control': self.read_reg(0x08),
            'status': self.read_reg(0x34)
        }
```

**2. 扫描测试模块 (`awg_uart_sweep.py`)**

```python
class AWGSweep:
    """AWG扫描测试"""

    def frequency_sweep(self, start_freq: float, end_freq: float,
                       points: int, duration: float):
        """频率扫描"""
        freqs = np.linspace(start_freq, end_freq, points)
        results = []

        for freq in freqs:
            self.controller.set_frequency(freq)
            time.sleep(duration)
            status = self.controller.read_status()
            results.append({
                'frequency': freq,
                'status': status
            })

        return results

    def amplitude_sweep(self, start_amp: float, end_amp: float,
                       points: int, duration: float):
        """幅度扫描"""
        amps = np.linspace(start_amp, end_amp, points)
        results = []

        for amp in amps:
            self.controller.set_amplitude(amp)
            time.sleep(duration)
            results.append({
                'amplitude': amp,
                'status': self.controller.read_status()
            })

        return results
```

- 停止位：1 bit
- 校验位：无

**命令格式：**

| 命令 | 格式 | 示例 | 响应 |
|------|------|------|------|
| 写寄存器 | `W <addr> <data>` | `W 08 00000003` | `OK` |
| 读寄存器 | `R <addr>` | `R 00` | `D 41574731` |
| 状态查询 | `S` | `S` | `STATUS: ...` |

**错误响应：**
- `E01` - 命令格式错误
- `E02` - 寄存器地址无效
- `E03` - 数据格式错误
- `E04` - 设备未就绪
