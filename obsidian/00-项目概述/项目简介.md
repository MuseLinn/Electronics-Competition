# 项目简介

目标是在正点原子 K7-325T 开发板上完成研电赛优利德赛题二的任意波形发生器 FPGA 数字基带。最终系统需要覆盖波形生成、幅度/偏置控制、扫频、存储、上位机控制和高速 DAC 输出。

## 赛题相关指标

| 指标 | 目标 |
|---|---|
| 采样率 | 不低于 5 GSa/s |
| 模拟带宽 | 不低于 1 GHz |
| 垂直分辨率 | 不低于 14 bit |
| 正弦最小频率 | 不高于 1 mHz |
| 最高频率 | 不低于 1 GHz |
| 频率分辨率 | 不高于 1 mHz |
| 谐波失真 | 优于 -40 dBc |
| 非谐波杂散 | 优于 -60 dBc |

## 当前工程策略

先把 FPGA 基础能力和工具链跑通，再做高速 DAC。

```text
License -> LED -> DDS -> 幅度/偏置 -> 扫频 -> BRAM/DDR3 -> AD9144 JESD204 -> PCIe 控制
```

当前最高优先级不是直接看示波器，而是先打通 AD9144 的数字链路。AD9144 链路必须先在 ILA 中看到 QPLL lock、TX reset done、TX ready、SYSREF、SYNC 和变化的数据，再判断模拟输出。

## 项目边界

- `D:\FPGA`：资料、standalone bring-up、AGENTS handoff。
- `D:\awg_fpga`：AWG 主工程、RTL、Vivado 工程、Obsidian vault。
- `D:\FPGA\ad9144_bringup_k325t`：AD9144 vendor demo 的 K325T standalone 验证包。
