# 硬件平台

## 主板

| 项目 | 规格 |
|---|---|
| 开发板 | 正点原子 K7-325T |
| FPGA | `XC7K325TFFG900-2I` |
| DDR3 | 4 x 4 Gbit NT5CC256M16，总计 2 GB |
| QSPI Flash | N25Q128，128 Mbit / 16 MB |
| 系统时钟 | 100 MHz 差分时钟 |
| PCIe | PCIe 2.0 x8 |
| FMC | HPC，连接到 Bank 117 GTX |

## DAC/ADC 子卡

| 项目 | 规格 |
|---|---|
| 子卡 | FMCADDA-9250-9144 |
| DAC | AD9144，16 bit，最高 2.8 GSa/s |
| ADC | AD9250，250 MSPS |
| 时钟芯片 | LMK04828 |
| 高速链路 | JESD204B，DAC 4 lanes，ADC 2 lanes |

## 关键注意

- AD9144 不在 K325T 主板上，它在 FMCADDA-9250-9144 子卡上。
- 正点原子教学 AD/DA 模块不是 K325T 主板集成外设；没有购买该模块时不要按教学 DAC 路线做板级验证。
- AD9144 目前的首要目标是证明 JESD204 链路接通，不是先追求最终 AWG 指标。

## 常用引脚

| 信号 | FPGA 引脚 | 说明 |
|---|---|---|
| `sys_clk_p` | AE10 | 100 MHz 差分时钟正端 |
| `sys_clk_n` | AF10 | 100 MHz 差分时钟负端 |
| `sys_rst_n` | AB25 | 复位按键，低有效 |
| `key[0]` | A26 | KEY0 |
| `key[1]` | A25 | KEY1 |
| `led[0]` | R24 | LED0 |
| `led[1]` | R23 | LED1 |

更多 FMC 引脚见 [[FMC引脚速查]]。
