# LMK04828 时钟配置

LMK04828 是 AD9144/JESD204 bring-up 的前置条件。它负责给 GTX/JESD 提供 refclk、global clock 和 SYSREF。

## 当前来源

当前不从零手写 LMK 配置，而是复用 vendor demo 中已经验证过的配置序列：

```text
D:\FPGA\FMCADDA-9250-9144\extracted_k7_full\fmcadda_9250_9144_demo_dac4L_k7\fmcadda_9250_9144.srcs\sources_1\new\lmk_spi_wr_config.v
```

## K325T FMC 低速映射

| LMK 信号 | FMC | FPGA 引脚 |
|---|---|---|
| `lmk_cs_n` | LA28_P | J11 |
| `lmk_sda` | LA28_N | J12 |
| `lmk_rst` | LA29_P | F15 |
| `lmk_sclk` | LA29_N | E16 |

注意：LMK SPI 使用 LA28/LA29，不使用 FMC IIC `AD29/AE29`。

## 相关时钟

| 信号 | FMC | FPGA 引脚 | 用途 |
|---|---|---|---|
| `refclk_p/n` | GBTCLK0_M2C | G8 / G7 | GTX refclk |
| `glblclk_p/n` | LA00_CC | D17 / D18 | JESD core/global clock |
| `sysref_p/n` | LA20 | D14 / C14 | JESD204 Subclass 1 |

## 调试顺序

1. 确认 FMCADDA 子卡已插稳。
2. 烧录 bitstream 后观察子卡 PLL 指示。
3. 在 ILA 中确认 QPLL lock。
4. 如果 QPLL 不 lock，优先检查 LMK SPI、refclk 引脚和时钟约束。
