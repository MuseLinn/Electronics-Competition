# 时钟树

## K325T 基础时钟

```text
sys_clk_p/n (AE10/AF10, 100 MHz differential)
  -> IBUFDS
  -> BUFG
  -> system clk
```

Verilog 必须使用差分输入缓冲：

```verilog
IBUFDS clk_ibufds (
    .I  (sys_clk_p),
    .IB (sys_clk_n),
    .O  (clk_ibuf)
);

BUFG clk_bufg (
    .I (clk_ibuf),
    .O (clk)
);
```

## AD9144/JESD 时钟

FMCADDA 子卡通过 LMK04828 提供 JESD 所需时钟：

```mermaid
flowchart TD
    LMK["LMK04828"] --> REF["GBTCLK0_M2C refclk G8/G7"]
    LMK --> GLB["glblclk D17/D18"]
    LMK --> SYSREF["SYSREF D14/C14"]
    REF --> GTX["GTXE2 QPLL/CPLL"]
    GLB --> JESDCORE["JESD204 core clocks"]
    SYSREF --> SUBCLASS["Subclass 1 deterministic latency"]
```
